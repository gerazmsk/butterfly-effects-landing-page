# Butterfly Effects - Deerfield Beach ABA Therapy Landing Page

A fully responsive, SEO-optimized landing page for the Butterfly Effects Deerfield Beach ABA Therapy Center.

## Features

- ✅ Fully responsive (mobile-first design)
- ✅ SEO-optimized for local keywords
- ✅ WCAG-compliant accessibility
- ✅ Modern, clean healthcare-inspired design
- ✅ FAQ accordion functionality
- ✅ No frameworks or dependencies

## Tech Stack

- HTML5 (semantic structure)
- CSS3 (custom stylesheet)
- Vanilla JavaScript (FAQ accordion only)

## Local Development

Simply open `index.html` in your browser or use a local server:

```bash
# Using Python
python3 -m http.server 8000

# Using Node.js (if you have http-server installed)
npx http-server
```

Then visit `http://localhost:8000`
